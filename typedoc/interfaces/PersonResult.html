Person Interface

Each result has:

  • id: person id
  • face: face object
  • body: body object
  • hands: array of hand objects
  • gestures: array of gestures
  • box: bounding box: x, y, width, height normalized to input image resolution
  • boxRaw: bounding box: x, y, width, height normalized to 0..1
